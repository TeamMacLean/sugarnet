# Sugar Net
<img align="right" height="128" width="128" src="https://raw.githubusercontent.com/wookoouk/sugarnet/master/public/img/logo.png">

### Depends on    
* [bower](http://bower.io)
* [bundler.io](http://bundler.io)
* [rails](http://rubyonrails.org)
* [R](http://r-project.org)

### Install
```
$ bundle install
$ bower install
```

### Start
```
$ rails server
```
